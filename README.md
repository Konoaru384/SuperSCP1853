> ⚠️ **Important Note**  
> SuperSCP1853 hooks into SCPSL’s item events to detect the AntiSCP207 + SCP-1853 combo and remove the Poisoned effect.  

> ⚠️ **Dependencies**  
> - Exiled **9.6.2**
---

## 🧩 Features

- 🛡️ Grants **immunity to poison** when both AntiSCP207 and SCP-1853 are used
